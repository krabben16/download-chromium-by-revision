# get-chromium-by-revision

This package downloads Chromium by specifying the revision number.

## usage

```shell
dcbr -r 727972
```

## options

```shell
Usage: cli [options]

Options:
  -r, --revision <number>  revision number
  -h, --help               display help for command
```

## develp

```shell
npm i
npm run dev
```
